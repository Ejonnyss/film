# Оснастка для Codex — что поставить для максимального качества

Дополнение к `CODEX_BRIEF_CLOUD_GPU_ARCANE_RU.md`. Здесь — конкретные движки,
библиотеки и интеграции, которые ощутимо поднимают качество фильма.
Всё бесплатное, кроме явно помеченного.

---

## 1. ГОЛОСА — главный запрос

Объём озвучки всего фильма: **2596 символов, 39 реплик, 4 роли**.
Это принципиально меняет расклад — текста так мало, что доступны даже премиум-сервисы
в рамках бесплатных лимитов.

### Рекомендуемый порядок

| Место | Движок | Цена | Русский | Комментарий |
|---|---|---|---|---|
| 🥇 | **ElevenLabs** | free 10 000 симв/мес — **наш фильм это 26% лимита** | отличный | Лучшее звучание на рынке, живые эмоции и дыхание. Весь фильм озвучивается бесплатно за один заход. Multilingual v2, голоса выбираются в Voice Library. |
| 🥈 | **Edge TTS** | бесплатно без лимита и ключа | очень хороший | Уже встроен в ноутбук (ячейка 14b). Нейронные голоса Microsoft. Разумный дефолт, если не хочется регистраций. |
| 🥉 | **XTTS v2 (Coqui)** | бесплатно, локально | хороший | **Клонирование.** Запишите по 15 секунд своего голоса — и Джон с Мэри заговорят вашими собственными. Для личного фильма это сильнее любого стокового голоса. |
| 4 | **Silero v4_ru** | бесплатно, офлайн | хороший | 5 тембров (aidar, baya, kseniya, xenia, eugene). Работает без интернета, на CPU, мгновенно. Запасной вариант. |
| 5 | **Higgsfield MCP** | платные кредиты | хороший | Уже подключён в Cowork: `generate_audio`, `create_voice`, `dubbing`. Имеет смысл, если кредиты всё равно останутся. |

### Как распределены роли (`voice_pack.json`)

| Роль | Реплик | Символов | Характер |
|---|---|---|---|
| Рассказчик | 23 | 1555 | Спокойный мужской, −12% темпа. Голос автора, не участник сцены. |
| Джон | 7 | 485 | Тёплый уверенный, с улыбкой в голосе, без напора. |
| Мэри | 6 | 395 | Живая, играющая, чуть дразнящая. |
| Мысли Мэри | 3 | 161 | Тот же тембр тише и ниже + реверберация. |

### Установка

```bash
pip install edge-tts                    # дефолт, уже в ноутбуке
pip install TTS                         # XTTS v2, клонирование
pip install elevenlabs                  # премиум, free tier
pip install silero                      # офлайн
```

ElevenLabs, минимальный пример под наш пак:

```python
from elevenlabs.client import ElevenLabs
import json
client = ElevenLabs(api_key="...")      # free-ключ из личного кабинета
vp = json.load(open("voice_pack.json", encoding="utf-8"))
VOICE_ID = {"narrator": "...", "john": "...", "mary": "...", "mary_inner": "..."}
for l in vp["lines"]:
    audio = client.text_to_speech.convert(
        voice_id=VOICE_ID[l["speaker"]],
        model_id="eleven_multilingual_v2",
        text=l["text"],
        voice_settings={"stability": 0.45, "similarity_boost": 0.8, "style": 0.35},
    )
    open(f"lines/{l['id']}.mp3", "wb").write(b"".join(audio))
```

Тайминги, обработка и сборка дорожки дальше берутся из ячеек 14c–14f ноутбука —
они не зависят от того, каким движком синтезирован звук.

## 2. МУЗЫКА

Романтическая тема на 5 минут. Варианты:

- **MusicGen (Meta, audiocraft)** — генерирует музыку по текстовому описанию, бесплатно,
  идёт на том же Colab-GPU. Промпт под наш фильм:
  `"slow cinematic romantic score, warm analog synth pads, soft piano, distant ocean,
  nostalgic, dreamy, 70 bpm, emotional, no drums"`
  ```bash
  pip install audiocraft
  ```
- **Stable Audio Open** — альтернатива, тоже бесплатная и открытая.
- **Готовые библиотеки:** Pixabay Music, Free Music Archive, Uppbeat — бесплатно
  под указание автора. Быстрее, чем генерировать.
- **Морской эмбиент:** freesound.org (CC0), искать `sea waves night calm`.

Уровни в миксе уже заданы: голос 0 dB, музыка −11 dB, море −13 dB, финал −14 LUFS.

---

## 3. ТОЧНЫЕ СУБТИТРЫ

Сейчас тайминги субтитров — оценка (14.5 знака в секунду). Чтобы они легли
идеально под реальную озвучку, нужно принудительное выравнивание:

```bash
pip install faster-whisper
# или
pip install whisperx
```

Прогнать готовую `voice.wav` через WhisperX с исходным текстом — получите
пословные тайминги и безупречный SRT. Это заметная разница в восприятии.

## 4. КАРТИНКА И ВИДЕО — что добавить сверх ноутбука

| Инструмент | Зачем |
|---|---|
| **Arcane LoRA** (Civitai) | Главный рычаг стиля. Ищите `arcane style SDXL LoRA`, вес 0.7–0.9. Без неё стиль держится только промптом. |
| **PuLID** | Альтернатива InstantID, часто даёт лучшее сходство на стилизации. Стоит попробовать обе и выбрать. |
| **RIFE** (уже в ноутбуке) | Интерполяция до 24 fps, убирает рывки в i2v-клипах. |
| **Real-ESRGAN / 4x-UltraSharp** | Апскейл кадров. Сейчас в ноутбуке простой LANCZOS — замена на модельный апскейл заметно поднимет детализацию. |
| **FaceFusion / ReActor** | Точечная починка лица, если в отдельном клипе «поплыло». |
| **CogVideoX-5B / Wan 2.1** | Запасные i2v-модели, если LTX-Video даст плохую стабильность лиц. |

## 5. ИНТЕГРАЦИИ И MCP

Для Codex как агента:

- **ElevenLabs MCP** — если хотите, чтобы Codex сам синтезировал реплики,
  а не через скрипт. Есть официальный сервер.

  создание голосов. Платные кредиты, но полезен для добивки отдельных сложных планов.
- **Google Drive** — Codex сможет забирать промежуточные файлы из Colab напрямую.
- **GitHub** — держать `11_CLOUD/` в репозитории: тогда ноутбук открывается в Colab
  по ссылке `colab.research.google.com/github/<user>/<repo>/blob/main/...ipynb`
  и не нужно каждый раз загружать файлы вручную. **Рекомендую сделать в первую очередь.**

## 6. ПОРЯДОК ДЕЙСТВИЙ ПО ЗВУКУ

1. Прогнать ячейку 14b на Edge TTS — получить рабочий вариант за 2 минуты.
2. Послушать. Если звучит достаточно — идти дальше, дорабатывать картинку.
3. Если хочется лучше — завести бесплатный ключ ElevenLabs и переозвучить
   (весь фильм = 26% месячного лимита), тайминги и микс не меняются.
4. Если хочется личного — записать свои голоса и прогнать XTTS v2.
5. Финально — WhisperX для точных субтитров, затем ячейка 14f (микс) и 15 (QC).

---

## 7. ЧЕГО НЕ ДЕЛАТЬ

- Не возвращаться к `macOS say` — именно от него уходим.
- Не давать рассказчику и Джону один тембр: зритель перестанет их различать.
- Не поднимать музыку выше −11 dB под речью — забьёт голос.
- Не синтезировать реплики Мэри и её мысли одинаково: внутренний голос обязан
  отличаться обработкой, иначе сцена читается как произнесённая вслух.
